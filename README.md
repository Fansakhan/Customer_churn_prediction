# Customer Churn Prediction Using Machine Learning

## Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. Acquiring new customers is often more expensive than retaining existing ones. Therefore, predicting which customers are likely to leave helps companies take proactive measures to improve customer retention.

This project develops a machine learning model that predicts customer churn using customer demographics, account information, service usage, billing details, and engineered features such as Customer Lifetime Value (LTV).

Multiple machine learning algorithms were trained and evaluated, and the best-performing model was selected based on prediction accuracy.

---

# Problem Statement

The objective of this project is to build a machine learning classification model that can accurately predict whether a customer will churn based on historical customer data.

The project aims to:

- Identify customers likely to churn.
- Discover important factors influencing customer churn.
- Help businesses improve customer retention strategies.
- Compare multiple machine learning algorithms to identify the best-performing model.

## 4. Feature Engineering

A new feature was created:

### Customer Lifetime Value (LTV)

Formula:

```
LTV = MonthlyCharges × Tenure
```

This feature estimates the overall revenue generated by each customer during their relationship with the company.

# Business Insights

The project identified several important insights:

- Customers with shorter tenure are more likely to churn.
- Month-to-month contracts have higher churn rates.
- Customers with higher monthly charges show increased churn probability.
- Customer Lifetime Value (LTV) is an important predictive feature.
- Contract type significantly influences customer retention.
- Customers using additional support services are less likely to churn.
